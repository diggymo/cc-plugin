# grep-sessions

Claude Code の全会話履歴から特定のキーワードで検索し、該当するセッションとマッチ内容を一覧表示するスキルプラグイン。

## 使い方

Claude Code 内で以下のように呼び出します:

```
/grep-sessions {キーワード}
```

## オプション

| オプション | 説明 |
|-----------|------|
| `--project {dir}` | 特定プロジェクトのみ検索 |
| `--limit {N}` | 表示件数の上限（デフォルト: 20） |

## 出力例

```
### session: abc12345-...  (2026-03-08 15:30)

マッチ箇所:
- ...キーワード周辺のスニペット...

直近のユーザー入力:
1. 最新の入力内容
2. その前の入力内容
3. さらにその前の入力内容

---
```

## ccs コマンド

ターミナルから直接使いたい場合は、以下の alias を `~/.zshrc` に追加してください:

```sh
alias ccs='python3 /path/to/plugins/grep-sessions/search.py'
```

## 仕組み

- `~/.claude/projects/` 配下の全 JSONL ファイル（subagents ディレクトリ含む）を検索
- 大文字小文字を区別しない
- 2パス検索: 1パス目でキーワードマッチ、2パス目でマッチしたセッションのユーザーメッセージを収集
- セッション ID 単位でユニーク化して表示
